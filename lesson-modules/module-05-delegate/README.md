# Module 05 — Delegate the Work

**~2 hours · 3 lessons**

---

## What this module teaches

Stop prompting steps. Start assigning goals.

This module teaches three patterns for delegating work to Claude:
- **Subagents** — specialist advisors who each have a distinct lens
- **Orchestrator** — one goal broken into pieces, handed to the right specialist
- **Agentic loops** — a closed loop that runs, verifies itself, fixes, and reruns

---

## The honest framing

These are patterns you build — not named Claude Code features you install.

The orchestrator is a prompt structure. The agentic loop is a goal + verification + retry instruction. Lessons 5.2 and 5.3 are explicit about this because the alternative (thinking it's a setting you toggle) leads to confusion and cost.
