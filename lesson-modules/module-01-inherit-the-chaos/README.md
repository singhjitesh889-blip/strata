## What you walk in with

- 11 ad CSVs named `FINAL_v2_USE_THIS_[n].csv`
- 200 lines of Hinglish ops chaos in a `.txt` file
- A GST sheet nobody opened since February

## What you walk out with

- One clean founder update — built in under 20 minutes
- Every number traced back to its source
- A `/founder-update` command you'll use every week
- Your first real Claude Code session: done

You're the new Growth Lead at **Nimbu Fresh** — a D2C juice startup in Bangalore.

The previous person quit last Friday. No handoff. You have their files.

The founder wants "one number on a slide" by Monday 10am.

It's Saturday afternoon. The files are in `../../inherited-chaos/`. Don't clean them up. That's the point.
