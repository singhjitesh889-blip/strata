# Lesson 2.3: One File, One Question

The WIKI layer sits between the raw data (11 messy CSVs) and the STATE (briefing doc).

Each WIKI file answers one category of question — and answers it better than any individual raw file could.

STOP: Ask me to show you the difference. First ask: "What was the ROAS for the Mango campaign?" — just as a question, no files.

## The Difference

"One answer was approximate. One was sourced. The WIKI file is what makes the second answer possible — always."

The WIKI layer isn't about organizing for its own sake. It's about making specific questions answerable with precision.
