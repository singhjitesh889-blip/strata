# Module 04 — Wire It to Live Data

**~1 hour · 2 lessons**

---

## What this module teaches

Until now, Claude has been reading files you manually loaded.

This module connects Claude to live data — so it can query your actual systems directly.

MCP (Model Context Protocol) gives Claude tools it doesn't have natively. Commerce data. SEO data. Whatever you connect.

---

## The honest part

MCP setup is where non-developers usually drop off. Lesson 4.1 is dedicated entirely to setup — not because it's hard, but because it has specific steps that fail in specific ways. Every failure mode is documented.
