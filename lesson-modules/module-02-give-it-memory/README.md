# Module 02 — Give It a Memory

**~2 hours · 5 lessons**

---

## The problem this solves

Every time you open a new Claude session, it knows nothing.

You've spent an hour in Module 01 explaining Nimbu Fresh. Monday morning you open Claude again — blank. You re-explain. Again. Every time.

This module ends that.

---

## What you build

A three-layer knowledge architecture for Nimbu Fresh:
- **RAW** — the mess (files, numbers, everything dumped)
- **WIKI** — organized knowledge (one file per question type)
- **STATE** — what Claude reads every session (≤800 tokens)
